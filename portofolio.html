Portofolio

Twitter clone

For my first-semester web development exam, we built a full-stack, minimal clone of X (Twitter) to explore the mechanics of a real-time social network. Our goal was to recreate core communication features, focusing on structured data handling and user relationships from the ground up.

We developed a responsive frontend using HTML, JavaScript, MojoCSS, and MixHTML. This was seamlessly connected to a Python and Jinja backend, utilizing a MySQL database to manage complex internal data flows, APIs, and a functional social feed.

Groent Marked

Grønt Marked was my final group project at KEA, where we designed and developed a dynamic website for a non-profit organization. After conducting client research and prototyping a complete design system in Figma, we brought the site to life using a modern, performance-driven tech stack.

We built the core framework with Astro.js and Tailwind CSS to ensure lightning-fast load times via partial hydration. For complex, dynamic features—like interactive filtering and state management, we integrated React.js and TypeScript for robust, type-safe code, fetching live database data to complete the user experience.

Caveman Studio

The project Cavman Studio was my 2. semester exam and a group project from KEA - Copenhagen School of Technology & Design. We collaborated to design a website for producer and musician Lucas Delacroix, founder of Caveman Studio.

The process started by doing comprehensive research and having a conversation with Lucas about his goals and requirements. We then moved into a design sprint, ultimately developing a complete design system and prototype in Figma. The site was built using the Astro framework, with Supabase providing database integration, and used JavaScript to fetch data from the database to make the website dynamic.

Additionally, we produced photos and videos for social media, which are also featured on the website.
